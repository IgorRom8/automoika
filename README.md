# 🚗 Сайт Автомойки

Простой сайт автомойки на Next.js с функциями регистрации и входа.

## Структура проекта

- **Главная страница** (`/`) - информация о услугах и ценах
- **Страница услуг** (`/services`) - полный список услуг
- **Регистрация** (`/register`) - создание нового аккаунта
- **Вход** (`/login`) - авторизация пользователя
- **Личный кабинет** (`/dashboard`) - профиль пользователя

## Запуск проекта

```bash
# Установка зависимостей (уже выполнено)
npm install

# Запуск в режиме разработки
npm run dev

# Открыть в браузере
http://localhost:3000
```

## Технологии

- Next.js 15
- TypeScript
- Tailwind CSS
- React

## Примечание

Это демо-проект. Аутентификация хранится в памяти и сбрасывается при перезапуске сервера. Для продакшена используйте базу данных и proper authentication (NextAuth.js, JWT и т.д.).
